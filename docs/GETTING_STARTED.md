# Getting Started

## Quick Start Guide
To get started with the Tricycle Cargo Bike project, clone the repository and install the necessary tools as outlined below.

## System Requirements
- Operating System: Windows, macOS, or Linux
- Python 3.7 or higher
- Node.js 14 or higher 
- Git version control system

## Tools Needed
1. **Text Editor / IDE**: VSCode, PyCharm, or any IDE of your choice.
2. **Node Package Manager (npm)**: To manage JavaScript packages.
3. **Python Environment**: Use `virtualenv` or `conda` to create isolated environments.

## Initial Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/boucettaabdelkhalek2-lgtm/Tricycle_Cargo_Bike_Original-1.git
   cd Tricycle_Cargo_Bike_Original-1
   ```
2. Set up Python environment:
   ```bash
   python -m venv env
   source env/bin/activate  # On Windows use `env\Scripts\activate`
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   npm install
   ```
4. Run the application:
   ```bash
   python app.py  # or your main entry point
   ```
5. Open your web browser and navigate to `http://localhost:5000` to see your application in action!
